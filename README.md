# FizeSecurity
PHP 安全库，为您的应用安全保驾护航。

FizeSecurity 是对一系列系统内置安全函数进行面向对象的再封装类库，同时添加了一些日常使用的安全类方法。 
FizeSecurity 是 Fize 项目族群的底层依赖，众多 Fize 项目依赖 FizeSecurity ，同时欢迎您使用 Fize。
FizeSecurity 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
